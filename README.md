# Miniproyecto 1 - Intercambio de regalos (Front-end)

Sistema web de intercambio de regalos desarrollado con HTML, CSS y JavaScript.
Incluye uso de APIs HTML5 (Drag & Drop) y LocalStorage para persistencia.

## Tecnologías
- HTML5
- CSS3
- JavaScript
- Bootstrap y/o Tailwind (según aplique)

## Funcionalidades
- Pantalla inicial con nombre y logotipo de la app.
- Captura del organizador y participantes (sin límite).
- Exclusiones entre participantes.
- Selección de tipo de evento y nombre personalizado.
- Selección de fecha del evento.
- Presupuesto del regalo.
- Vista de datos del evento leyendo desde LocalStorage (no desde RAM).
- Sorteo (diseño original) respetando exclusiones.
- Implementación de Drag & Drop con propósito dentro del sistema.
- Responsive design con mínimo 2 breakpoints usando grid (Bootstrap/Tailwind).
